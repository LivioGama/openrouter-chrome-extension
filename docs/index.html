Privacy Policy for OpenRouter CSV Cost Analyzer

Overview

The OpenRouter CSV Cost Analyzer is a Chrome extension that helps users analyze their AI usage costs on the OpenRouter platform. This privacy policy explains how we handle data when you use our extension.

Information We Collect

Data Processing

  • Activity Data: The extension processes your OpenRouter activity data (usage costs, token counts, and model information) that you have already authorized OpenRouter to access
  • Local Storage: Temporary caching of processed data for up to 30 minutes to improve performance
  • No Personal Information: We do not collect, store, or transmit any personal information, email addresses, passwords, or other user credentials

API Communications

  • OpenRouter API: The extension makes direct API calls to OpenRouter's servers using your existing authenticated session
  • No External Servers: All data processing happens locally in your browser
  • No Data Storage: No user data is sent to or stored on external servers

How We Use Information

Functionality

  • Cost Analysis: Process your activity data to calculate and display cost breakdowns by AI model
  • Performance Optimization: Cache processed results temporarily to reduce API calls and improve user experience
  • User Interface: Display analysis results with interactive charts and sorting options

Data Handling

  • Local Processing: All data analysis happens in your browser
  • Session-Based: Data is only processed during your active OpenRouter session
  • User Control: You can clear cached data anytime using browser developer tools

Data Sharing and Disclosure

No Third-Party Sharing

  • We do not share, sell, or rent any user data to third parties
  • No data is transmitted outside of your OpenRouter session
  • All processing remains within your browser environment

OpenRouter's Privacy

  • Your activity data remains subject to OpenRouter's own privacy policy
  • The extension only accesses data you have already authorized OpenRouter to collect

Data Security

Browser-Based Processing

  • All sensitive operations happen locally in your browser
  • No data transmission to external servers
  • Your OpenRouter authentication remains secure

Cache Management

  • Temporary data is stored only in browser localStorage
  • Cache automatically expires after 30 minutes
  • You can manually clear cache using developer console commands

User Rights and Controls

Data Access

  • You have full control over your OpenRouter data
  • The extension only processes data you choose to analyze
  • You can disable or remove the extension at any time

Cache Control

  • Clear cached data anytime with: OpenRouterDev.clearCache() in browser console
  • Cache is automatically cleared when you close your browser
  • No persistent data storage

Children's Privacy

This extension is not intended for children under 13. We do not knowingly collect any information from children under 13.

Changes to This Privacy Policy

We may update this privacy policy from time to time. We will notify users of any material changes by updating the date at the top of this policy.
